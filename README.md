# BoardingBook

A full-stack boarding place management system for students, built with React + Express + MongoDB.

## Project Structure

```
BoardingBook/
├── frontend/       # React + Vite frontend (port 5173)
├── backend/        # Express + MongoDB backend (port 5001)
└── docs/           # Project documentation
```

## Getting Started

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

**Backend**
```bash
cd backend
npm install
npm run dev
```
