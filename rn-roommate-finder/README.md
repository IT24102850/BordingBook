# Roommate Finder App

This is a React Native application designed to help users find roommates and form groups. The app allows users to create profiles, browse available roommate profiles, send requests, and manage group formations.

## Features

- **User Profiles**: Users can create and manage their profiles, including personal details and preferences.
- **Roommate Matching**: Browse through available roommate profiles and find potential matches based on shared interests and preferences.
- **Group Formation**: Users can create and join groups, facilitating easier communication and collaboration among roommates.
- **Notifications**: Receive notifications for roommate requests and group invitations.

## Installation

To get started with the Roommate Finder app, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd rn-roommate-finder
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the application:
   ```
   npm run start
   ```

## Usage

- Launch the app on your mobile device or emulator.
- Create a user profile to start browsing for roommates.
- Use the navigation to access different screens, including browsing profiles and managing group requests.
